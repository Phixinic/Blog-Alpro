Menghitung Luas Persegi

⇛Deskripsi Algoritma

  1. Mulai
  2. Masukkan panjang sisi persegi (s)
  3. Hitung luas persegi dengan rumus: Luas = s x s
  4. Tampilkan hasil luas persegi.
  5. Selesai

Menentukan Bilangan Ganjil atau Genap

⇛Deskripsi Algoritma

  1. Mulai
  2. Masukkan bilangan bulat x.
  3. Hitung sisa pembagian x dengan 2: Sisa = x mod 2.
  4. Jika sisa = 0, maka bilangan adalah genap. Jika sisa ≠ 0, maka bilangan adalah ganjil.
  5. Tampilkan hasil.
  6. Selesai
