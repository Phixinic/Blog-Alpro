Menentukan Bilangan Terbesar dari 3 Bilangan

>Deskripsi Algoritma

  1. Mulai
  2. Masukkan 3 bilangan: A, B dan C
      3. Jika A >= B dan A >= C, maka A adalah bilangan terbesar.
      Jika B >= A dan B >= C, maka B adalah bilangan terbesar.
      Jika tidak, maka C adalah bilangan terbesar
  3. Tampilan hasil bilang terbesar
  4. Selesai
